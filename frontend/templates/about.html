MangaArr

Manga, Manwa, and Comics Collection Manager

Version 1.0.0

About MangaArr

MangaArr is a manga, manwa, and comics collection manager with a focus on release tracking and calendar functionality. It follows the design principles of the *arr suite of applications but with a specialized focus on manga and comics.

This project was created as a fork of the retired Readarr project, focusing specifically on manga and comics while removing some features and adding others.

MangaArr is open source software, and contributions are welcome!

Key Features
  • Interactive release calendar for manga and comics
  • Track your manga/comic collection
  • Monitor upcoming releases
  • Home Assistant integration
  • Homarr integration
  • Modern, responsive web interface
