Home Assistant Integration

Integrate MangaArr with your Home Assistant instance

Home Assistant is an open source home automation platform that puts local control and privacy first.

With this integration, you can:

  • View upcoming manga/comic releases in your Home Assistant dashboard
  • Receive notifications for new releases
  • Monitor MangaArr status

REST API Configuration

Add the following to your Home Assistant configuration.yaml:

sensor:
  - platform: rest
    name: mangarr_upcoming_releases
    resource: http://your-mangarr-host:7227/api/integrations/home-assistant
    value_template: "{{ value_json.stats.series_count }}"
    json_attributes_path: "$"
    json_attributes:
      - upcoming_releases
      - stats
    scan_interval: 3600
Lovelace Card Example

Add a custom card to your dashboard:

type: entities
title: MangaArr
icon: mdi:book-open-variant
entities:
  - entity: sensor.mangarr_upcoming_releases
    name: Series Count
    icon: mdi:bookshelf
  - type: attribute
    entity: sensor.mangarr_upcoming_releases
    attribute: stats.volume_count
    name: Volume Count
    icon: mdi:book-multiple
  - type: attribute
    entity: sensor.mangarr_upcoming_releases
    attribute: stats.chapter_count
    name: Chapter Count
    icon: mdi:file-document-multiple

Homarr Integration

Integrate MangaArr with your Homarr dashboard

Homarr is a sleek, modern dashboard that puts all your apps and services in one place.

With this integration, you can:

  • View MangaArr status in your Homarr dashboard
  • See today's releases at a glance
  • Quick access to MangaArr interface

Homarr Configuration
  1. In your Homarr dashboard, click "Edit Layout"
  2. Click "Add Integration" and select "Generic App"
  3. Fill in the following details:
    • Name: MangaArr
    • Icon: fas fa-book
    • URL: http://your-mangarr-host:7227
    • Status Endpoint: /api/integrations/homarr
  4. Save your changes

Homarr will now display the status of your MangaArr instance and show today's releases.

API Documentation

MangaArr provides a comprehensive REST API that can be used to integrate with other services.

Below are some of the key API endpoints:

Endpoint Method Description
/api/series GET Get all series
/api/series/{id} GET Get a specific series with volumes and chapters
/api/calendar GET Get calendar events
/api/calendar/refresh POST Refresh calendar data
/api/integrations/home-assistant GET Get data for Home Assistant integration
/api/integrations/homarr GET Get data for Homarr integration

For a complete API documentation, refer to the API Documentation.
